### 🧭 项目总览

我构建了一系列技术驱动的在线工具和信息聚合平台，专注于数据可视化、实时信息展示和 SEO 优化。

| 项目 | 简介 | 核心技术与亮点 | 访问链接 |
| :--- | :--- | :--- | :--- |
| **每日60s速览** | 新闻聚合与天气可视化，支持多城市动态切换。 | 边缘计算 (Edge Computing)、Serverless 函数、Chart.js、JSON-LD 结构化数据 | [🔗 访问](https://60s.quanxin365.cn/) |
| **每日金价观察站** | 实时贵金属价格看板，覆盖国内国际、银行及品牌报价。 | 边缘计算、API 缓存策略、服务端渲染 (SSR)、数据可视化 | [🔗 访问](https://jinjia.quanxin365.cn/) |
| **金融量化监控平台** | 黄金/白银/美元汇率走势分析，布林带技术指标，金银比价计算。 | Python、静态站点生成 (SSG)、趋势算法、自适应布林带模型 | [🔗 访问](https://jinrong.quanxin365.cn/) |
| **号卡优选商城** | 电话卡办理咨询服务，多运营商套餐对比。 | 静态 HTML、语义化标签、FAQ 结构化数据 | [🔗 访问](https://haoka.quanxin365.cn/) |
| **权芯365 · 博客** | 记录日常，分享生活的个人博客。 | React、TypeScript、TailwindCSS、服务端渲染 (SSR) | [🔗 访问](https://blog.quanxin365.cn/) |
| **权芯365 · 主页** | 个人主页/导航，展示所有技术作品。 | 静态 HTML、ItemList 结构化数据 | [🔗 访问](https://www.quanxin365.cn/) |
